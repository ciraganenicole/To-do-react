# Todo-react-app

This is an exercise where I built a todo list using resct

## Built With

- REACT
- CSS

## Getting Started


To get a local copy up and :

- Open your terminal and cd to the folder where you want to put the project and run :
 ```
 git clone git@github.com:ciraganenicole/To-do-react.git
 ```

### Prerequisites

- Create-react-app
- Git/Github
- VSCode or another equivalent tool
- Html, Css and Javascript
- Webpack
- APIs
- Jest

### Install

```
npm install
```
```
npm install --save-dev eslint@7.x eslint-config-airbnb@18.x eslint-plugin-import@2.x eslint-plugin-jsx-a11y@6.x eslint-plugin-react@7.x eslint-plugin-react-hooks@4.x @babel/eslint-parser@7.x @babel/core@7.x  @babel/plugin-syntax-jsx@7.x @babel/preset-env@7.x  @babel/preset-react@7.x
```
```
npm install --save-dev stylelint@13.x stylelint-scss@3.x stylelint-config-standard@21.x stylelint-csstree-validator@1.x
```

### Run tests

Run the following tests:

- npm start 
- npm run test
- npx stylelint "**/*.{css,scss}"
- npx eslint .
